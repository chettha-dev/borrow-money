## Installation

```bash
$ npm install
$ cp env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Using docker-compose

```bash
# build docker container
$ docker-compose up -d
```

## API Document

```bash
# open browser with url
$ http://localhost:3000/document
```
